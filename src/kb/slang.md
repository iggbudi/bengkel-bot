# BengkelBot — Slang & Bahasa Pemetaan

## Semarang/Jawa slang yang sering dipakai pelanggan → terjemahan formal

| Slang | Terjemahan | Contoh |
|---|---|---|
| wes / udh / udah | sudah | "wes service tapi masih bunyi" |
| mas / bang | sapaan informal (pria) | "mas, mobil saya bunyi" |
| mbak | sapaan informal (wanita) | "mbak, tune up berapa?" |
| ndak / nggak / kagak | tidak | "ndak ada bunyi waktu rem" |
| iso / bisa | bisa | "iso booking jam 9 pagi?" |
| arep / mo h mau | akan / ingin | "arep service lengkap" |
| ono / ada | ada | "ono bunyi dari depan" |
| kok / koq | mengapa | "kok mahal banget?" |
| to / yo / ya | ya / iya | "yo, mau booking" |
| sak / se | satu | "sak mobil" |
| gek / sek | tunggu | "gek aku tanya dulu" |
| nopo | apa | "nopo kui?" |
| ra / ora | tidak (Jawa) | "ra tau kui" |
| ben | supaya / agar | "ben iso" |
| ae / wae | saja | "booking ae" |
| ki / kui | ini / itu | "ki masalahnya" |
| bang / mas | sapaan teman sebaya | "bang, mau tanya" |

## Ekspresi khas Semarang

| Ekspresi | Arti |
|---|---|
| "mas" | sapaan untuk laki-laki (mirip "bang") |
| "cok" | ekspresi kekagetan/konfirmasi |
| "wes" | sudah |
| "mau kemana mas" | mau kemana (kalimat khas Semarang) |
| "yo ndak" | ya nggak / bisa jadi |
| "gak sih" | mungkin tidak  |
| "wes ae" | sudah saja |
| "gaskeun" | semangat (dipengaruhi Sunda/Bandung) |

## Cara bot merespons

- Kalau pelanggan pakai slang → balas pakai slang juga (tapi sopan)
- Jangan gunakan slang berlebihan — tetap bisa dipahami
- Contoh: "Oke mas/ mbak, wes diterima bookingnya! Nanti jam 9 pagi ya 👋"
